# ANSI-Escape-Colors

Shows all ANSI Escape Colors from 0-100

This is also my first C++ project, hope you like it

# Installation

`git clone https://github.com/iinc0gnit0/ANSI-Escape-Colors`

`cd ANSI-Escape-Colors`

`chmod +x main`

# Usage

`./main`
